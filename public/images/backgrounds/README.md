# Background Images

Add these Spider-Verse themed images to create an animated, immersive background:

## Required Images

1. **spiderman-city-swing.png**
   - Miles Morales swinging through the city (Spider-Verse style)
   - Will be used as the main parallax background
   - Recommended: 1920x1080px or larger
   - Will be scaled and animated with parallax

2. **miles-gwen.png**
   - Miles and Gwen floating together (Spider-Verse style)
   - Will float in top-right area
   - Recommended: 800x1200px
   - Animated with floating and rotation effects

3. **silhouette-fire.png**
   - Stylized silhouette with fiery/glowing elements
   - Will be positioned bottom-left
   - Recommended: 600x800px
   - Animated with pulsing glow effects

4. **baseball-character.png**
   - Character with baseball bat (glitch art style)
   - Will float in the middle-left area
   - Recommended: 400x600px
   - Animated with rotation and movement

## Animation Features

- **Parallax Scrolling**: Images move at different speeds as you scroll
- **Floating Effects**: Smooth up/down and rotation animations
- **Glow Effects**: Pulsing brightness and drop shadows
- **Opacity Fades**: Images fade based on scroll position
- **Web Line Overlays**: Animated web lines sweep across the background

## File Placement

Place all images directly in this directory:
```
public/images/backgrounds/
├── spiderman-city-swing.png
├── miles-gwen.png
├── silhouette-fire.png
└── baseball-character.png
```

The portfolio will work without these images, but they enhance the visual experience significantly!
